# random_select

Flutter-приложение на Dart: выбирает случайных персонажей для 6 игроков.

- Данные персонажей хранятся в SQLite.
- Имена игроков и количество ролей хранятся в SQLite.
- Для каждого игрока выдается 2 персонажа на выбор.
- Количество танков, ДД и хилов можно менять в приложении, общий лимит - 6 игроков.
- Выбор показывается анимированным колесом удачи.

## Запуск

```bash
flutter create .
flutter pub get
flutter run
```

Если нужно изменить список персонажей, отредактируй seed-данные в `lib/data/hero_seed_data.dart`
или удали базу приложения, чтобы SQLite пересоздалась с новым списком.
